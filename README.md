##Vitry
A JavaScript-based music composition environment

### Requirements
- [Java SE 6](http://www.java.com/en/download/) or better
- [Rhino](http://www.mozilla.org/rhino/)
- [Apache Ant](http://ant.apache.org/) (for building)

### Build
For complete build:
<pre>
ant
</pre>

For options:
<pre>
ant -p
</pre>
